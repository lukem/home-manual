# Artifacts

This `Artifacts/` directory contains non-Markdown artifacts used by the
Home Manual.

Contents could include:

- Diagrams/figures:
  - `.drawio`: [draw.io](https://app.diagrams.net) source files and `.png` exports.
  - `.svg`: SVG files from other tools.
- Configuration backups of various equipment. Remember to add the suffix to `.gitattributes`.
